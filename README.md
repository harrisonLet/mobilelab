# mobilelabs

A Python toolkit for processing and analyzing mobile laboratory atmospheric measurement data.

## Overview

`mobilelab` provides preprocessing utilities and analysis algorithms for working with data collected from mobile atmospheric sensing platforms. It is designed for use in processing large multi-instrument datasets.

## Installation

Not ready for use


## Modules

### `mobilelab.preprocess`
- **`preprocess`** — Reads and standardizes raw instrument data files, cleaning bad values and removes superfluous data columns
- **`merge`** — Aligns and merges data streams from multiple instruments

### `mobilelab.algorithms`
- **`plume`** — Plume identification routine
